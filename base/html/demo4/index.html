张杰

中国大陆男歌手。2004年参加第一届“我型我秀”获总冠军出道，2005年推出首张专辑《第一张》并凭借成名曲《北斗星的爱》获得全球华语榜中榜“年度传媒推荐大奖”等奖项， 2007参加湖南卫视快乐男声获得全国第四名，除歌唱事业，张杰2012年捐资200万作为北斗星空爱心基金创始基金，启动“张杰音乐梦想教室”公益项目，并获得“中国儿童慈善奖—-突出贡献奖”。至2014年6月已陆续在全国各地建立了二十七所“张杰音乐梦想教室”。首位登上美国洛杉矶Dolby Theatre（杜比剧院）举行个人演唱会的中国大陆艺人。

经历

  1. 2005年，推出首张专辑《第一张》，并凭借成名曲《北斗星的爱》获得全球华语歌曲排行榜最受欢迎新人奖、TVB全球华语歌曲金曲榜最佳男新人金奖等多个奖项。
  2. 2006年，发行第二张专辑《再爱我一回》，并首次涉足影视，担任校园电影《化学反应》主演之一。之后，由于经纪公司高层人事变动事业陷入低谷。
  3. 2007年，参加湖南卫视选秀节目快乐男声，以发片歌手的身份重新参加选秀并成为快乐男声成都赛区冠军和全国第四名的成绩，后来签约天娱传媒。被媒体视为中国大陆华语歌坛新生代领军人物之一，在中国大陆年轻族群具有较高人气。
  4. 2008年5月1日，在北京举办首场个人演唱会。[2]；同年10月，推出专辑《明天过后》，并凭借该专辑获得第六届特步东南劲爆音乐榜的内地最受欢迎男歌手奖。

捐款门事件

2008年，张杰转投天娱，并向原公司提出解约。为此，张杰的粉丝发起了一个募捐活动，为他筹措105万元解约金。最终，这场解约以张杰赔款告终。2012年5月2日，一位自称曾是张杰粉丝的网友在大陆天涯论坛踢爆张杰“私吞”粉丝募款，而当时承诺报答粉丝的不售票演唱会却只字不提。5月5日，一位网友还晒出了当年粉丝筹集100万元的汇款单以及公证书原件的照片，该事件迅速引发热议。随后张杰发表声明称，粉丝募集的解约金未经自己手。不久后，某知名论坛上出现爆料帖，矛头直指选秀艺人，文中将“粉头”用来召集粉丝买票、集资的证据一一列出，“受之鼓动的年轻粉丝，为了偶像可以不吃不喝、不眠不休，但资金却不知去向”[17]。最终，张杰在有力证据下承认事实并道歉[18]。
